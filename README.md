# commitcheckup

TypeScript ile yazılmış bir npm paketi.

## Kurulum

```bash
npm install commitcheckup
```

## Kullanım

```typescript
import { greet } from 'commitcheckup';

console.log(greet('Dünya')); // "Hello, Dünya!"
```

## Geliştirme

1. Projeyi klonlayın
2. Bağımlılıkları yükleyin: `npm install`
3. Geliştirme yapın
4. Build alın: `npm run build`
